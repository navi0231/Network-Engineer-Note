### 9.2.1 ipconfig
<details><summary><font size="5"> 内容 </summary>

> - ipconfig是Windows系统最常见的网络管理工具，可以显示所有网卡的TCP/IP配置
- ipconfig的语法如下：
```
 ipconfig [/all] [/renew [Adapter]] [/release [Adapter]]
         [/flushdns] [/displaydns] [/registerdns]
         [/showclassid Adapter] [/setclassid Adapter [ClassID]]
```
- 以上参数具体解读如下：

|参数|具体解释|
|---:|---:|
|/?|显示帮助信息|
|/all|显示所有网卡的完整 TCP/IP 配置信息|
|/renew [*Adapter*]|更新指定网卡的 DHCP 配置|
|/release [*Adapter*]|释放指定网卡的 IPv4 地址，断开 DHCP 租约|
|/flushdns|**刷新并清除客户端 DNS 缓存内容**|
|/displaydns|显示当前 DNS 缓存中的内容|
|/registerdns|刷新所有 DHCP 租约并重新注册 DNS 名字|
|/showclassid *Adapter*|显示网卡的 DHCP 类别 ID|
|/setclassid *Adapter* [*ClassID*]|对指定的网卡(Adapter)设置DHCP类别ID|

- 具体实例：
```

### 9.2.2 ping
<details><summary><font size="5"> 内容 </summary>

> ping通过发送[ICMP回声](https://github.com/navi0231/network-engineer-note/blob/main/02-Internet-connection/5.2.4ICMP-message-types.md)请求报文来检验与另外一个计算机的连接

- ping的语法如下：
```
ping [-t] [-a] [-n Count] [-l Size] [-f]
     [-i TTL] [-v ToS] [-r Count]
     [-s Count] [{-j HostList | -k HostList]
     [-w Timeout] [-TargetName]
```

- 命令参数解读如下：

|参数|具体解释|
|---:|---:|
|-t|持续发送回声直到被中断|
|-a|反向解析|
|-n *Count*|指定发送回声次数|
|-l *Size*|发送回声的字节数|
|-f|force极限测试，测试最大报文长度|
|-i *TTL*|指定IP头中TTL的值|
|-v *ToS*|指定IP头中ToS的值|
|-r *Count*|在IP头中添加路由记录|
|-s *Count*|在IP头中添加时间戳|
|-j *HostList*|在IP头中使用松散路由选项|
|-k *HostList*|在IP头中使用严格路由选项|
|-w *Timeout*|指定回声响应时间|
|TargetName|用IP或主机名指定目标名称|

- 具体实例：
```
